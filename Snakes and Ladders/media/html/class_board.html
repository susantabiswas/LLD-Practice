Detailed Description

Class to represent the board. It is a dim x dim dimensioned board. It can have special game objects like Snakes and Ladders.

addGameObject(): Adds special game elements reachedEnd(player): whether the player has finished the game or not movePlayer(player, int moves): Returns the final position of player after moves steps printBoard(): Prints the board
